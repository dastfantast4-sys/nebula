# nebula
# NEBULA — минималистичный магазин мебели

Тёмный одностраничный магазин с мгновенной фильтрацией, живым поиском, корзиной и модальными карточками товаров. 

![stack](https://img.shields.io/badge/stack-HTML%20%C2%B7%20CSS%20%C2%B7%20JS-2d8a5e)

## Фишки

- **Мгновенная фильтрация** по категориям — с плавным fade-out / fade-in, без перезагрузки.
- **Живой поиск** с debounce и транслитерацией: «атлас» найдёт «Диван Atlas».
- **Модальные карточки товаров** — клик по любой карточке открывает полноэкранное окно с фото, описанием, материалами, размерами и кнопкой покупки.
- **Корзина** — выезжающая панель справа: количество, изменение qty, сумма, сохранение в `localStorage`.
- **Микро-взаимодействия**: товар «улетает» в корзину, бейдж пульсирует, карточки поднимаются при наведении и появляются со stagger-задержкой.
- **Дизайн-система**: CSS-переменные, сетка на `auto-fill minmax`, один акцентный цвет (`#2d8a5e` — тёмно-зелёный), отступы кратны 8, единый easing `cubic-bezier(0.4,0,0.2,1)`.
- **Адаптив**: от десктопа 1280px до мобильного (2 колонки карточек, полноэкранная корзина, модалка в одну колонку).
- **Единая предметная съёмка**: все 20 фото приведены к одному тёмному студийному стилю с зелёным контровым светом.

## Структура
├── index.html      # разметка (включая модальное окно товара)
├── styles.css      # дизайн-система, тёмно-зелёная тема, стили модалки
├── script.js       # каталог, фильтры, поиск, корзина, модальные окна
├── data.js         # массив из 20 товаров с описаниями
└── assets/img/     # фото товаров (1024×1024)
